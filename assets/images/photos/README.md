# Organisation des Photos pour Sarah

Ce dossier contient les photos que Sarah peut envoyer en réponse aux demandes des utilisateurs.

## Structure des noms de fichiers

Les photos doivent être nommées en incluant les mots-clés correspondant à leur catégorie. Par exemple :

- `cul_1.jpg`, `fesses_2.jpg` pour les photos de derrière
- `poitrine_1.jpg`, `seins_2.jpg` pour les photos de poitrine
- `jambes_1.jpg`, `cuisses_2.jpg` pour les photos de jambes
- `visage_1.jpg`, `face_2.jpg` pour les photos de visage
- `corps_1.jpg`, `body_2.jpg` pour les photos de corps entier
- `lingerie_1.jpg`, `sousvetements_2.jpg` pour les photos en lingerie
- `bikini_1.jpg`, `maillot_2.jpg` pour les photos en bikini
- `selfie_1.jpg`, `photo_2.jpg` pour les selfies

## Catégories disponibles

1. **Cul/Fesses** : `cul`, `fesses`, `derriere`, `butt`, `ass`
2. **Poitrine** : `poitrine`, `seins`, `boobs`, `tits`, `chest`
3. **Jambes** : `jambes`, `cuisses`, `thighs`, `legs`
4. **Visage** : `visage`, `face`, `tete`, `head`
5. **Corps** : `corps`, `body`, `full`, `entier`
6. **Lingerie** : `lingerie`, `sous-vetements`, `underwear`
7. **Bikini** : `bikini`, `maillot`, `swimsuit`
8. **Selfie** : `selfie`, `photo`, `picture`

## Format des fichiers

- Formats acceptés : `.jpg`, `.jpeg`, `.png`
- Taille recommandée : 1080x1920 pixels (format portrait)
- Taille maximale : 16MB

## Conseils

1. Utilisez des numéros séquentiels pour chaque catégorie
2. Évitez les espaces dans les noms de fichiers
3. Utilisez des underscores (_) pour séparer les mots
4. Assurez-vous que les photos sont de bonne qualité
5. Respectez les droits d'auteur et la vie privée
